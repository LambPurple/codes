### C++的八股文

1. 为什么空类占用1字节的大小？
即使一个类没有成员变量，它也会占用至少 1 字节的内存空间，
以确保每个对象都有唯一的地址，并满足可能的对齐要求。

2. 谈谈shared_ptr<void>?
shared_ptr<void> 是 C++ 中的一种特殊情况，
表示一个可以管理任意类型动态分配内存的智能指针。

3. 引用折叠,完美转发
```c++
//在这个类中，对左值引用，右值引用和常量引用做出的行为是一致的
class Stock{
public:
    template <typename T>
    explicit Stock(T &&arg) : name(std::forward<T>(arg)),price(0){}
    ...

private:
    std::string name;
    int price;
}

```
我们使用了一个模板构造函数，使用了完美转发（std::forward<T>(s)）来接受字符串参数 s。这个构造函数可以接受左值引用、右值引用和常量引用，并根据传递的参数类型进行相应的转发。


4.如何将this指针变成一个shared_ptr?
当一个类继承自 std::enable_shared_from_this 时，它会获得一个成员函数 shared_from_this()，该函数返回一个 std::shared_ptr，指向调用它的对象。这个 std::shared_ptr 与通过直接创建 std::shared_ptr 使用 new 或 std::make_shared 创建的指针不同，它们共享相同的引用计数和控制块。
